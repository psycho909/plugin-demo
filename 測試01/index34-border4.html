📏 頂部藍色區域 + 垂直線條拉伸

你的圖片特點：

  • 🔵 頂部有藍色區域
  • 📏 有垂直分隔線（左、中、右）
  • ⬇️ 需要垂直拉伸，讓線條跟著拉長

解決方案：使用 border-image 的 stretch 模式！

🎯 9-Slice 切片示意圖

關鍵：左右邊框會垂直拉伸，所以垂直線條會跟著高度增長！

⬇️ 拖動底部調整高度

📝 完整 HTML + CSS 代碼

推薦代碼（最簡潔）：

<!-- HTML --> <div class="border-box"> <h2>標題</h2> <p>你的內容...</p> </div>
/* CSS - 最簡潔版本 */ .border-box { /* 設定邊框寬度 */ border: 30px solid transparent; /* 使用圖片，stretch 模式（所有方向都拉伸） */ border-image: url('border-image2.png') 30 stretch; /* 內容樣式 */ padding: 40px; background: white; background-clip: padding-box; /* 尺寸控制 - 只限制寬度，讓高度自由增長 */ width: 500px; min-height: 400px; }

完整控制版本：

/* CSS - 完整分解版本 */ .border-box { /* 邊框基本設定 */ border-width: 30px; border-style: solid; /* border-image 屬性分解 */ border-image-source: url('border-image2.png'); /* 切片：從圖片邊緣切30px作為邊框 */ /* 加 fill 關鍵字會顯示中間區域 */ border-image-slice: 30 fill; /* 邊框實際顯示寬度 */ border-image-width: 30px; /* 重複模式：stretch = 拉伸 */ border-image-repeat: stretch; /* 內容樣式 */ padding: 40px; /* 尺寸控制 */ width: 500px; min-height: 400px; }

🔑 關鍵參數說明

border-image-slice: 30

  • 📏 從圖片四個邊各切 30px 作為邊框
  • 🔵 這個數值決定藍色區域的高度
  • 💡 如果藍色區域看起來太高或太矮，調整這個數值
  • ✨ 加上 fill 關鍵字會顯示中間區域

border-image-repeat: stretch

  • ↔️ 水平方向：藍色區域拉伸以適應寬度
  • ⬇️ 垂直方向：垂直線條拉伸以適應高度
  • 🎯 這正是你需要的效果！

background-clip: padding-box

  • 🎨 確保背景色不會覆蓋邊框
  • ✅ 讓邊框圖片清晰顯示
  • 💡 這是讓效果完美的關鍵設定

❓ 常見問題解答

Q1: 藍色區域高度不對怎麼辦？

A: 調整 border-image-slice 的數值。

  • 如果藍色太高 → 減少數值（例如改成 25）
  • 如果藍色太矮 → 增加數值（例如改成 35）

Q2: 垂直線條位置不準確？

A: 確保原始圖片中的垂直線是對稱的，且在正確的位置。

💡 線條的位置由原始圖片決定，CSS 只負責拉伸。

Q3: 想要固定寬度，只拉伸高度？

A: 設定固定 width，只用 min-height：

.border-box { width: 500px; /* 固定寬度 */ min-height: 400px; /* 最小高度，可以更高 */ }

✅ 最佳實踐建議

  1. 圖片準備：
    • 確保圖片是標準的 9-slice 格式
    • 四個角的大小一致
    • 使用 PNG 格式保留透明度
  2. CSS 設定：
    • 固定寬度，讓高度自由增長
    • 使用 stretch 模式
    • 加上 background-clip: padding-box;
  3. 測試：
    • 在不同高度下測試效果
    • 確保在各種尺寸下都完美
    • 檢查不同瀏覽器的兼容性
