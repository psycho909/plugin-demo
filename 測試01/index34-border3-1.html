📝 HTML + CSS 代碼

基本用法：

<!-- HTML --> <div class="border-box"> <p>你的內容</p> </div>
/* CSS */ .border-box { /* 設定邊框寬度 */ border: 30px solid transparent; /* 使用你的圖片作為邊框 */ border-image: url('向量圖智慧型物件.png') 30 stretch; /* 內容區域背景 */ padding: 40px; background: white; background-clip: padding-box; }

完整控制（進階）：

.border-box { border: 30px solid transparent; /* 分解的寫法 */ border-image-source: url('向量圖智慧型物件.png'); border-image-slice: 30; /* 切片位置 */ border-image-width: 30px; /* 邊框寬度 */ border-image-repeat: stretch; /* stretch | round | repeat */ padding: 40px; background: white; background-clip: padding-box; }

✅ 關鍵參數說明

  • border-image-slice: 30 - 從圖片邊緣切 30px 作為邊框（根據你的圖片調整）
  • border-image-repeat: stretch - 拉伸模式，適合你的邊框
  • background-clip: padding-box - 確保背景不會覆蓋邊框

⚠️ 注意事項

  • 確保 border-image-slice 的值與你圖片的邊框寬度匹配
  • 如果圓角看起來不完美，調整 slice 數值
  • 圖片需要是正方形或矩形，且邊框均勻分布效果最佳
  • 使用 PNG 格式可以保留透明度
